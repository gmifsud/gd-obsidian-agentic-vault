*   **Language:** Markdown (essential syntax covered: headings, paragraphs, bold, italics, strikethrough, ordered/unordered lists, task lists, code blocks, tables).
*   **Obsidian Features/Commands:**
    *   `Split Right` (for multi-pane view)
    *   `Ctrl/Cmd + O` (Open existing file)
    *   `Ctrl/Cmd + B` (Bold), `Ctrl/Cmd + I` (Italics)
    *   `[[Internal Link]]` (double square brackets)
    *   `[External Link Text](URL)`
    *   `![[Image Link]]` and `![[Image Link|Size]]` (image embedding and resizing)
    *   Code blocks and inline code (using backticks)
    *   Tables (manual Markdown or right-click insert)
    *   Callouts (`>[!type] Title`)
    *   Source Mode, Editing Mode, Reading View
    *   `#tag` (inline tags)
    *   YAML (properties block at top of note: `--- tags: [tag1, tag2] ---`)
    *   Backlinks (visible in right sidebar)
    *   Embedded Search Query (`query tag:obsidian` within code blocks)
    *   Block References (`[[Note Name^block-id]]` or `![[Note Name^block-id]]`)
*   **Community Plugins:**
    *   **DataView**: Demonstrated for creating dynamic Maps of Contents (MOCs).
*   **Note-taking Methodologies/Concepts:**
    *   Zettelkasten System
    *   PARA (Projects, Areas, Resources, Archives)
    *   Pillars, Pipes, and Votes
    *   Knowledge, Action, Explorations
    *   Fleeting Notes, Literature Notes, Distilled/Permanent Notes
    *   Knowledge Garden (Seedling, Growth, Evergreen, Maintenance stages)
    *   Maps of Contents (MOCs)
    *   Emoji Tag Taxonomies (inspired by Brian Janks)
    *   Note Prefixes (e.g., `CAN-` for Canvas, `TV-` for YouTube videos)

## Key Learning Points
*   **Master Fundamental Markdown:** Learn to format text efficiently using Obsidian's Markdown syntax for clear and readable notes.
*   **Build Interconnected Knowledge:** Understand and apply internal links, external links, and block embeds to create a rich network of interconnected ideas, improving understanding and recall.
*   **Effective Note Categorization:** Utilize both inline tags and structured YAML properties for robust categorization, enabling advanced search and organization of your vault.
*   **Leverage Backlinks for Discovery:** Actively use the backlinks pane to discover how different notes relate to each other, forming a more holistic understanding of your knowledge.
*   **Implement Maps of Contents (MOCs):** Learn to create MOCs (static or dynamic with DataView) as central index pages to navigate and organize notes around specific topics or themes.
*   **Adopt a Strategic Note-Making Cycle:** Embrace the "Knowledge Garden" metaphor, actively cultivating notes through seedling, growth, and evergreen stages, emphasizing continuous review, refinement, and connection.
*   **Explore Advanced Obsidian Capabilities:** Experiment with features like callouts for emphasis, embedded search queries for contextual references, and the DataView plugin for powerful dynamic note generation to enhance your PKM system.

## Key Message
The core message is to move beyond basic note-taking in Obsidian by mastering its foundational Markdown and powerful linking capabilities, and then applying structured methodologies like Maps of Contents (MOCs) and a "Knowledge Garden" approach. This enables the cultivation of a highly interconnected, searchable, and evolving personal knowledge management system that significantly enhances learning, organization, and recall.
