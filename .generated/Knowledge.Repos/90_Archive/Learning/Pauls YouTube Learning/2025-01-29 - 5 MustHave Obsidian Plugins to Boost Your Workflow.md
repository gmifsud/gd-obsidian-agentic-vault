This reduces context switching and creates a unified workspace, proving more efficient for specific, pre-configured sites than Obsidian's native Web Viewer.
3.  **Automated Note Organization by Tags (Auto Note Mover):** Automatically move notes to specific folders based on predefined tag rules. This hands-free organization method saves time and maintains a tidy vault, especially useful for managing notes as they evolve or are first created.
4.  **Tag-Driven Navigation and Refinement (Tag Folder):** Navigate and view notes organized by tags as if they were folders in the left pane. This provides a powerful alternative or complement to traditional folder structures, making it easier to find notes related to a specific topic or to "flush out" notes in a particular development stage (e.g., "seedling" notes).
5.  **Interactive Data Capture with Modal Forms:** Create custom, interactive pop-up forms for structured data entry (e.g., daily notes, book reviews, project forms). This ensures consistency in information capture, integrates well with templates, and simplifies data input processes.
6.  **Efficient Tag Management (Tag Wrangler):** Quickly rename, merge, toggle, and search tags directly from the tag pane. This is crucial for maintaining a clean, organized, and consistent tag system as your vault grows, facilitating easier knowledge retrieval and refactoring.
7.  **Concept of Evolving Note States:** Utilizing tags like 'seedling' (new, undeveloped), 'distilled' (refined), and 'evergreen' (permanent, stable) to denote the lifecycle and development stage of a note. This provides a clear framework for working through information and promoting knowledge growth.
