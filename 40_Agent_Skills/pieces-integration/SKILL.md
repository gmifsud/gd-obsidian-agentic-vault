# SKILL: Pieces Integration

**Objective:** Interact with the Pieces OS daemon to manage code snippets, extract context, and save reusable components.

**Execution:**
1. When generating a reusable code component or discovering a useful snippet, use the Pieces MCP server to save it.
2. When asked to retrieve a snippet, query the Pieces MCP server.
3. Ensure that any saved snippet includes relevant tags (e.g., `react`, `tailwind`, `agentic-os`) for easy retrieval.
